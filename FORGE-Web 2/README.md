# FORGE — Système de Progression Personnel

> *Deviens celui que tu veux être.*

Web app PWA installable sur iPhone et Mac. Aucun serveur, aucune dépendance — fonctionne hors ligne.

## Démo

👉 **[forge-app.github.io](https://ton-username.github.io/forge-app)**

Login démo : `demo` / `demo123`

## Fonctionnalités

Dashboard · Habitudes · Timer Focus Pomodoro · Sport & Programme · Calendrier · Scanner EDT IA · Forge Pass · Cercle · Rangs · Abonnement

## Installer sur iPhone

1. Ouvre le lien dans **Safari**
2. Appuie sur **Partager** → **Sur l'écran d'accueil**
3. FORGE apparaît comme une vraie app native

## Installer sur Mac

1. Ouvre dans **Chrome ou Edge**
2. Clique l'icône **⊕** dans la barre d'adresse → **Installer FORGE**

## Stack

Vanilla HTML · CSS · JS — aucun framework, aucun build tool.
